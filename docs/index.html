01 Premise

The reference the family ports to.

Mainsail is the canonical Python implementation. Easy to embed, easy to read, easy to extend — drop a module into mainsail/applets/ with the four-symbol contract (NAME, ALIASES, HELP, main) and auto-discovery picks it up.

The applet roster, flag conventions, exit codes, and stdout byte semantics defined here are what the sister projects verify against. The tradeoff is interpreter cold-start — what it gives back is a 148 KB portable .pyz that runs anywhere Python 3.8+ does, and the easiest codebase in the family to fork and bend.

03 Quickstart

Native binary · pipx · zipapp.

From a release, pipx, source, or a portable .pyz you can scp anywhere Python 3.8+ runs.

# From a release — no Python required
$ curl -LO https://github.com/Real-Fruit-Snacks/mainsail/releases/latest/download/mainsail-linux-x64
$ chmod +x mainsail-linux-x64
$ ./mainsail-linux-x64 --list

# From source — Python 3.10+
$ git clone https://github.com/Real-Fruit-Snacks/mainsail && cd mainsail
$ pip install -e .
$ mainsail --list

# Or via pipx — isolated venv, mainsail on PATH
$ pipx install git+https://github.com/Real-Fruit-Snacks/mainsail.git

# Wire up your shell
$ mainsail install-aliases ~/.local/bin                  # symlink ls/cat/grep/...
$ mainsail completions bash | sudo tee /etc/bash_completion.d/mainsail
$ mainsail update                                          # self-update

# Portable zipapp — runs anywhere Python 3.8+ does
$ scp mainsail.pyz host:/tmp/
$ ssh host 'python3 /tmp/mainsail.pyz find /var/log -mtime -7'

05 Architecture

Four files, auto-discovered applets.

Adding an applet means dropping a module into mainsail/applets/ with the four-symbol contract. registry.py's pkgutil.iter_modules picks it up on next launch — no registration table to edit.

mainsail/
__main__.py     // entry → cli.main(argv)
cli.py          // dispatch · argv[0] basename + subcommand
registry.py     // auto-discovery · NAME + ALIASES → main
usage.py        // per-applet --help
common.py       // shared helpers
applets/        // 84 modules
  ls.py         // NAME, ALIASES, HELP, main(argv) -> int
  cat.py
  …

Stdin via sys.stdin.buffer for binary safety; stdout via sys.stdout.buffer when bytes are required. Exit codes: 0 success · 1 runtime · 2 usage.
