FEATURES

Everything you reach for, in one place

A familiar POSIX surface. Real flag coverage, not stubs. Same behavior on Windows, macOS, and Linux.

One Binary, Fifty-One Utilities

Every common shell tool dispatched through a single executable. Multi-call mode lets you hardlink to applet names so ls just works.

$ mainsail --list | wc -l
51
$ ln mainsail ls && ./ls -la
~5MB
Native Binary

Native Windows

Runs on bare Windows — no WSL, no Cygwin, no git-bash. Recognizes dir, type, copy, del, where as native aliases.

Real Applets

find with -exec, -prune, parens, boolean ops. sed with addresses and in-place edit. awk with BEGIN/END, patterns, arrays, gsub. tar with compression filters.

Pipeline-Grade I/O

Binary-safe through cat/tee/gzip. tail -f with rotation detection. xargs -0 for paths with backslashes.

Cross-Platform Integrity

Same SHA-256 of "abc" on every platform. tar archives interchangeable across OS. 23-case stress harness verifies behavior on Windows, Linux, and macOS.

Hackable

ARCHITECTURE

Four-layer dispatch

Clean separation between the entry point, dispatch logic, applet registry, and per-applet implementations. Adding an applet means dropping one file.

1

Entry

__main__.py (or the installed mainsail script) hands sys.argv to cli.main(). The frozen Nuitka binary uses the same entry — no separate path.

2

Dispatch

cli.py resolves the applet from argv[0] basename (multi-call) or argv[1] (subcommand). Intercepts --help only; -h stays free for applet flags like df -h.

3

Registry

registry.py lazy-loads mainsail.applets.* via pkgutil.iter_modules on first call. Each module exposes NAME, ALIASES, HELP, and main(argv) -> int.

4

Applets

~50 modules in mainsail/applets/, ~30–250 lines each. Read binary via sys.stdin.buffer; write via sys.stdout / sys.stdout.buffer. Return 0 / 1 / 2 (success / runtime / usage).

GETTING STARTED

Install or build

Pre-built binaries on the releases page. Install from PyPI for hackability. Build from source for the full standalone experience.

install.sh
# From source
$ git clone https://github.com/Real-Fruit-Snacks/mainsail.git
$ cd mainsail && pip install -e .

$ mainsail --version
mainsail 0.1.13

# Or freeze a standalone binary
$ pip install "Nuitka[onefile]"
$ python build.py
→ dist/mainsail (~5 MB, no Python at runtime)

# Or a portable zipapp — runs on any Python 3.8+
# (ESXi, exotic arches, restricted hosts)
$ python build.py --pyz
→ dist/mainsail.pyz (~80 KB)

# Or a custom subset — pick exactly the applets
# you want; presets: slim, minimal
$ python build.py --pyz --applets ls,cat,grep,sed,awk
→ dist/mainsail-custom.pyz (~18 KB)
usage.sh
# Daily-driver pipelines
$ mainsail find . -name '*.py' -mtime -7

# Cross-platform integrity
$ mainsail sha256sum *.tar.gz

# Real sed
$ mainsail sed -i 's/foo/bar/g' *.txt

# Windows-native aliases
$ mainsail dir .                   :: == ls
$ mainsail copy a.txt b.txt        :: == cp
